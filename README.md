# Awesome Skills

A personal collection of useful agent skills 😉.

## Installation

### GitHub Copilot

```bash
gh skills install gerrytan/awesome-skills azurerm-acctest --agent github-copilot --scope user
```

### Claude Code

```text
/plugin marketplace add gerrytan/awesome-skills
/plugin install azurerm-acctest@gerrytan-awesome-skills
```
